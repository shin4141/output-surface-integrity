# Evidence Boundary

[Back to the README RTK section](../README.md#rtk-and-osi-use-the-level-the-work-requires)

This directory contains selected preserved research evidence and public-facing schemas. Executable RTK artifacts are not run by OSI, are not required for Level 1 adoption, and are not installation artifacts.

The authoritative RTK project is the [upstream RTK repository](https://github.com/rtk-ai/rtk). Preserved research files must not be treated as current RTK releases. Their identity and hashes are retained only for research traceability, and their presence supports no OSI-versus-RTK winner claim.
