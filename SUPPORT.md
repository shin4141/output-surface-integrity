# Support

Use GitHub Issues for:

- usage questions;
- documentation problems;
- reproducible defects; and
- claim or evidence inconsistencies.

Start with [Level 1](docs/quick-use.md) for ordinary use. Choose the closest issue form and provide bounded, reproducible evidence.

This is independent open-source work with no service-level agreement. Research-design consultation and emergency operational support are not guaranteed.

Do not post secrets, private logs, personal data, private evidence, internal identifiers, or private filesystem paths publicly. Follow [SECURITY.md](SECURITY.md) for sensitive reports.
